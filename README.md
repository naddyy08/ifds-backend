# 🖥️ My E-Portfolio - Backend (API)

This is the server-side application that powers my E-Portfolio, handling data storage and API requests.

## 🔗 Live API Base URL
`https://ifds-backend.onrender.com/`

## 🚀 Features
* **RESTful API:** Clean endpoints for project data and contact forms.
* **Security:** CORS enabled for frontend communication.
* **Database:** Integrated with MongoDB (or your choice).

## 🛠️ Tech Stack
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB Atlas

## ⚙️ Setup Instructions
1. Clone the repo: `git clone https://github.com/naddyy08/ifds-backend.git`
2. Install dependencies: `npm install`
3. Add your `DATABASE_URL` and `PORT` to a `.env` file.
4. Start the server: `npm run dev`
